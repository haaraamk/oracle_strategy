# 🔮 나스닥 하이브리드 오라클 전략 대시보드

개인 교육용 백테스팅 & 시장 모니터링 도구  
**투자 권유 아님 — 과거 수익률은 미래를 보장하지 않습니다**

---

## 🚀 설치 & 실행 (3단계)

```bash
# 1. 패키지 설치
pip install -r requirements.txt

# 2. 실행
streamlit run app.py

# 3. 브라우저에서 자동으로 열림
#    → http://localhost:8501
```

---

## 📱 주요 기능

### 탭 1 — 백테스트 결과
- QQQ DCA vs 오라클 전략 10년 성과 비교
- 스위칭 타점(진입/청산) 시각화
- CAGR, MDD, 샤프 비율 비교
- 실제 스위칭 이력 테이블

### 탭 2 — 현재 시장 상태
- 3개 필터 실시간 충족 여부 (게이지 표시)
- 오늘 기준 신호 상태 확인
- 최근 90일 지표 추이 차트

### 탭 3 — 시뮬레이터
- 월 적립금 + 목표일 입력
- 전략 CAGR 기반 예상 자산 계산
- 목표 달성도 시각화

## ⚙️ 전략 파라미터 (사이드바에서 실시간 조정)

| 파라미터 | 기본값 | 설명 |
|---|---|---|
| VIX 임계값 | 25 | 이 값 이상이면 공포 구간 진입 |
| QQQ 모멘텀 기준 | 2% | 20일 수익률이 이 값 이상이면 추세 전환 확인 |
| TNX 60일 한도 | 5% | 10년물 금리 60일 변화율이 이 값 미만이면 유동성 안전 |

---

## ⚠️ 주의사항

1. **Overfitting 주의**: 파라미터를 과거 데이터에 맞게 조정하면 결과가 좋아 보임
2. **변동성 잠식**: SOXL 같은 3× 레버리지 ETF는 장기 보유 시 지수의 3배보다 수익이 낮음
3. **미반영 비용**: 세금(양도소득세 22%), 환전비용, 슬리피지 미포함
4. **데이터 의존성**: 인터넷 연결 필요 (Yahoo Finance)
